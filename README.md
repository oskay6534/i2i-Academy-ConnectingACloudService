# i2i Academy - Connecting A Cloud Service - 1

Bu proje, i2i Academy staj programı kapsamında **Google Cloud Platform (GCP)** üzerinde temel bir bulut altyapısı (Virtual Machine) kurmak, SSH ile güvenli bağlantı sağlamak ve ağ bağlantısını (ping testi) doğrulamak amacıyla gerçekleştirilmiştir.

## Yapılan İşlemler
1. **GCP Compute Engine** üzerinde Ubuntu 26.04 LTS Minimal işletim sistemine sahip bir sanal makine (instance) başarıyla ayağa kaldırılmıştır.
2. Tarayıcı tabanlı SSH protokolü kullanılarak uzak sunucuya güvenli erişim sağlanmıştır.
3. Linux terminali üzerinden `echo` ve `cat` komutları kullanılarak ödevde istenen dosya yönetimi (dosya oluşturma ve okuma) işlemleri tamamlanmıştır.
4. Güvenlik Duvarı (Firewall) yapılandırması kontrol edilerek yerel bilgisayardan bulut sunucusuna internet üzerinden erişim (Ping testi) başarıyla gerçekleştirilmiştir.
